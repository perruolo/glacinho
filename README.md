# 🐧 Glacinho

> Seu companheiro polar de autocontrole.

Glacinho é um app motivacional e de produtividade pessoal com tema polar. Você cuida de um pinguim virtual enquanto constrói hábitos, organiza tarefas, registra sessões de foco e sobe de nível.

## ✨ Funcionalidades

- **Pinguim virtual** com stats (energia, felicidade, disciplina, limpeza), estados de humor e interações com cooldown
- **Hábitos** com streak, XP dobrado após 7 dias consecutivos e consistência diária
- **Tarefas** com prioridade e XP reward
- **Cronômetro** livre e **Pomodoro** (25 min foco + 5 min pausa automática)
- **Intenção do dia** com confirmação e recompensa de XP
- **Sistema de XP e níveis** (8 níveis, do Filhote de Gelo ao Espírito do Gelo Eterno)
- **16 conquistas** desbloqueáveis
- **Reflexão diária** salva por data
- Visual polar com aurora boreal, neve e estrelas animadas

## 🚀 Como usar

Nenhuma instalação necessária. Abra o `index.html` no navegador ou acesse pelo link do GitHub Pages.

Todos os dados ficam salvos localmente no navegador (localStorage) — funciona 100% offline.

## 🛠 Stack

- HTML + CSS + JavaScript puro (zero dependências)
- SVG animado
- localStorage para persistência
